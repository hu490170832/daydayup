## 正则

### 正则字符

`^`：匹配输入字符串的开始位置。

`$`：匹配输入字符串的结束位置。如果

`*`：匹配前面的子表达式零次或多次,例如，zo_ 能匹配 "z" 以及 "zoo"。_等价于{0,}。

```js
/a*/.test('zoo'); //true
```

`+`：匹配前面的子表达式一次或多次。例如，'zo+' 能匹配 "zo" 以及 "zoo"，但不能匹配 "z"。+ 等价于 {1,}。

`?`：匹配前面的子表达式零次或一次。例如，"do\(es\)?" 可以匹配 "do" 或 "does" 中的"do" 。? 等价于 {0,1}。

**`{n,m}`**：m 和 n 均为非负整数，其中n &lt;= m。最少匹配 n 次且最多匹配 m 次。
